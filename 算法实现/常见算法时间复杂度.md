# 优劣情况
>[!TIP|label:时间复杂度按数量级递增排列顺序]
>常数阶O(1)、对数阶O(log2n)、线性阶O(n)、线性对数阶O(nlog2n)、平方阶O(n2)、立方阶O(n3)、……k次方阶O(nk)、指数阶O(2n)。 

```
优<----------------------<劣
O(1)<O(㏒2n)<O(n)<O(n2)<O(2n)
```
